# SimplLibraryCRM

Таблица Person
```
create table Person (
    id int GENERATED BY DEFAULT AS IDENTITY,
    name varchar NOT NULL,
    birth_year int CHECK (birth_year > 1900) NOT NULL,
    phone varchar CHECK (length(phone) = 12) UNIQUE,
    email varchar UNIQUE
)
```
